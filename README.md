DDoS Simulation and Mitigation Tool

Description

This project is a Python-based simulation tool that demonstrates how DDoS attacks can overload a system and how basic mitigation techniques like rate limiting can help control traffic.

Features

Simulates incoming network requests

Detects system overload condition

Implements basic rate limiting concept

Demonstrates simple DDoS mitigation strategy

Technology Used

Python

Problem Statement

Distributed Denial of Service (DDoS) attacks can overload a system by sending a large number of requests. This project simulates such a scenario and demonstrates how rate limiting can help prevent system failure.

How to Run the Program

Open the project folder in Visual Studio Code.

Open the terminal.

Run the command:

python ddos_simulator.py

Sample Output

Request received: 1 Request received: 2 Request received: 3 ... System overload detected Mitigation applied: Rate limiting activated

Simulation finished
